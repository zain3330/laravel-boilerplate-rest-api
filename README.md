# Laravel API Boilerplate - Auth + User CRUD

This is a lightweight **Laravel API boilerplate** with:

- **Sanctum Authentication** (Register, Login, Logout)
- **UserController CRUD** (Create, Read, Update, Delete)
- **Role & Permission Management** using [Spatie Laravel-Permission](https://spatie.be/docs/laravel-permission/v5/introduction)
- Middleware protection for routes
- API-ready structure to reuse in multiple projects
